A Sudoku board that can be represented in three different ways:

1. Linear Sequential Board - All entries are stored linearly in one array.   [Entry0-81]
2. Row-Column Board - Entries are stored by row and column.   [row0-8][col0-8]
3. Block-based Board - Each entry is stored in its respective block.   [block0-8][entry0-8]

Plans:
     -Turn this into an actual CLI game, allowing you to fill the board entry by entry.
     -Make difficulty options for randomly generated boards.
